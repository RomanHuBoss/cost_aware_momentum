# Patch 1.2.1

Исправляет ситуацию, когда после перехода на dynamic universe на экране оставались только 5 демонстрационных символов.

## Причины

1. Поле Bybit `symbolType` ошибочно использовалось как crypto/non-crypto classifier, хотя это регион/сегмент пары.
2. Уже завершенный hourly inference текущего часа не запускался повторно после расширения universe.
3. UI/API имели ограничение выдачи, которое мешало отображению полного списка.

## После установки

```powershell
py -3.12 manage.py doctor
py -3.12 manage.py run
```

Миграция PostgreSQL не требуется. После первоначальной загрузки в верхней строке интерфейса появится: `Universe: N из M · динамический · карточек K`.
